# Quest Chains Mono Repo

## Available Scripts

In the project directory, you can run:

### Dapp

#### `yarn dapp:dev`

#### `yarn dapp:build`

#### `yarn dapp:start`

### Contracts

#### `yarn contracts:compile`

#### `yarn contracts:test`

#### `yarn contracts:deploy`

### Subgraph

#### `yarn subgraph:codegen`

#### `yarn subgraph:build`
